## Installation

### Cloning The Repository

```bash
git clone https://github.com/marchewadm/portfolio.git
cd portfolio
```

#### Via Docker Compose

1. **Build Docker Image**

```bash
docker compose build
```

2. **Run The Container**

```bash
docker compose up -d
```

3. **Stop The Container**

```bash
docker compose down
```

#### Without Docker

1. **Install All dependencies**

```bash
npm install
```

2. **Run Development Server**

```bash
npm run dev
```
